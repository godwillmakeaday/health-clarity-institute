# Health Clarity Institute Concept Brief PDF - Clean Version

This package contains a PDF concept brief with no Vercel URL or temporary deployment domain.

## Included file

- `public/docs/health-clarity-institute-concept-brief.pdf`

## Install in Termux

```bash
cd ~/projects/health-clarity-institute
unzip -o /sdcard/Download/health-clarity-concept-brief-clean-web-package.zip
git add public/docs/health-clarity-institute-concept-brief.pdf README-CONCEPT-BRIEF-PDF-CLEAN.md
git commit -m "Add clean Health Clarity concept brief PDF"
git pull --rebase origin main
git push origin main
```

After deployment and production promotion if needed, the PDF should be available at:

`/docs/health-clarity-institute-concept-brief.pdf`

Note: The PDF intentionally omits the temporary Vercel URL. Official domain/contact details can be added after final domain activation.
