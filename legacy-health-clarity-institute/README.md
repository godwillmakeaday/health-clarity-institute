# Archived: Health Clarity Institute

This directory contains the previous project that lived at the root of this
repository — a Nigerian health-education and patient-navigation platform
(Next.js App Router, TypeScript, Tailwind).

It has been moved here, unmodified, to make room for **The Verified Income
Desk**, which now occupies `app/`, `components/`, `data/` and `lib/` at the
repository root.

This code is kept for future reference and possible revival as a separate
project. It is **not** wired into the active Next.js app — the root
`app/` directory now belongs to The Verified Income Desk, and Tailwind's
`content` globs no longer scan this folder. If this project is revived, it
should likely move to its own repository or its own root-layout route group.
